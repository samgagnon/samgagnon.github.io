Problem-solving Strategies

  1. Draw a diagram if appropriate
  2. Determine which quantities are known and which are unknown
  3. Solve using variable before substituting values
  4. Verify units/dimensions
  5. Check limiting/special cases
  6. For numerical results: check the order of magnitude
